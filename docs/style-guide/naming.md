# 命名风格指南

## 文件名称

1. 第一原则： 使用各个编程语言社区的规范（Rust, Python, JavaScript, Shell, Solidity）
2. 第二原则： 与 [google-style] 保持一致

[google-style]: https://developers.google.com/style/filenames
